# Handwriting recognition :pen:


## Prerequisites

- Python 3 :snake: or later
- pipenv, to install it use `$ pip install pipenv` _(optional)_
#### <small> You can use any virtualenv tool, use the requirements.txt file to install requirements.</small>

## Usage
- Install requirements using:
```cosnole
$ pipenv install
or
$ pip install -r requirements.txt
```
- Download the dataset from [here](https://www.kaggle.com/crawford/emnist) (Balanced dataset) and put each `csv` file in `data/emnist`

- Run the notebook

- You'll get a `h5` model ready to use
